# InstaPic Client

## Production URL

https://desolate-taiga-13604.herokuapp.com

## Run the app locally


### `yarn`

Installs required packages.

### `yarn start`

Runs the app in the development mode.<br />
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The app makes request to [instapic-server](https://github.com/janroguska/instapic-server)

Set the request url in the `.env` file found in the root directory.

### `yarn build`

Builds the app for production to the `build` folder.

### `yarn test`

Runs tests.

### `yarn lint`

Lints code.
